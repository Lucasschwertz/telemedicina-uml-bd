# Plataforma de Telemedicina

Projeto desenvolvido para a disciplina **Práticas Extensionistas III** do curso de **Análise e Desenvolvimento de Sistemas - UNOESC**.

## Objetivo

Desenvolver uma plataforma de telemedicina capaz de centralizar o gerenciamento de atendimentos médicos, permitindo o cadastro de usuários, autenticação, consultas e controle das informações clínicas em um único ambiente.

## Problema Identificado

O acesso aos serviços médicos ainda apresenta desafios relacionados à organização dos atendimentos, agendamento de consultas, centralização de informações dos pacientes e acesso remoto aos serviços de saúde.

## Solução Proposta

A solução consiste em uma plataforma web de telemedicina que permite:

* Cadastro de usuários;
* Controle de acesso por login;
* Gerenciamento de pacientes;
* Gerenciamento de médicos;
* Controle de consultas;
* Registro de prontuários;
* Pesquisa e consulta de informações;
* Formulário de contato com os desenvolvedores.

## Funcionalidades Implementadas

### Interface Principal

Página inicial da plataforma contendo acesso às funcionalidades principais.

### Autenticação

Tela de login para controle de acesso dos usuários.

### CRUD

Operações de:

* Cadastro
* Consulta
* Alteração
* Exclusão

### Pesquisa

Filtros e mecanismos de busca para localização de registros.

### Formulário de Contato

Canal para comunicação entre usuários e desenvolvedores da plataforma.
